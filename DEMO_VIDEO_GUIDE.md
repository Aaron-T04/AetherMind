## ğŸ¥ Recording Tips

### Screen Recording Software
**Windows:**
- **OBS Studio** (free, professional) - Recommended
- **Windows Game Bar** (Win + G) - Built-in, simple
- **Camtasia** (paid, easy editing)

**Mac:**
- **QuickTime** (built-in, simple)
- **OBS Studio** (free, professional)
- **ScreenFlow** (paid, easy editing)

### Setup Before Recording

1. **Clean Browser:**
   - Close unnecessary tabs
   - Use incognito/private mode if needed
   - Zoom browser to 100-125% for clarity

2. **Prepare Workflow:**
   - Test the workflow once before recording
   - Make sure API keys are working
   - Have a good research topic ready

3. **Audio:**
   - Use a good microphone or headset
   - Record in a quiet room
   - Speak clearly and at a moderate pace

4. **Visual:**
   - Record at 1080p (1920x1080) minimum
   - Frame rate: 30fps is fine, 60fps if possible
   - Make sure text is readable
